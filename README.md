# 🎨 Drawing App (PyQt)

A GUI-based drawing application built using Python and PyQt.

---

## 🚀 Features

* ✏️ Freehand drawing
* 🧽 Eraser
* 🎨 Color picker
* 📏 Brush size
* ⬛ Rectangle and ⚪ circle shapes
* 📝 Text tool
* 🌈 Rainbow brush (dynamic color mode)
* 💾 Save drawings as image

---

## 🛠️ Tech Stack

* Python
* PyQt5

---

## ▶️ Run

pip install PyQt5
python draw_app.py
